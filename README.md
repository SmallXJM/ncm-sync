# ncm-sync

ncm-sync 是一个**个人音乐库自动化同步工具**，目前仍处于早期开发阶段，仅用于技术交流学习、研究使用。

> ⚠️ **重要说明**：本项目不提供、不内置任何绕过版权保护、付费限制或技术保护措施的功能，仅对**用户已购买或已获得合法授权的音乐内容**进行自动化同步。
