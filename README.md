# ncm-sync

ncm-sync 是一个音乐同步工具，目前仍处于早期开发阶段。

## 贡献与社区

- 欢迎提交 Issue 与 Pull Request
- 请勿提交敏感信息（如Cookie、账号密码等）
- 在提出功能建议时，尽量描述清楚场景与预期
